# octopusdeploy_platform_hub_version_control_username_password_settings (Resource)

This resource manages Platform Hub version control settings with username and password authentication in Octopus Deploy.

~> Credentials are stored in state as plaintext. [Read more about sensitive data in state.](https://www.terraform.io/language/state/sensitive-data)

## Example Usage

```terraform
resource "octopusdeploy_platform_hub_version_control_username_password_settings" "example" {
  url            = "https://github.com/acme/hello-world.git"
  default_branch = "main"
  base_path      = ".octopus"
  username       = "git-user"
  password       = "###########" # get from secure environment/store
}
```
